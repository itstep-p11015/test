1. Александров


На клетчатом поле 2Х2, 3х3 или 4х4 (можно настроить) расположены различные символы. Время  Время от времени рядом с полем
 появляется какой-нибудь символ. Если такой же есть на поле, то пользователь должен как можно быстрее нажать на соответствующий символ на поле, если нет - то не нажимать. За промахи предусмотрены штрафы.

2. Дикий

На ферме пушистых зверьков ведётся экономический учёт - доход от продажи сырья, обслуживание зданий, зарплаты сотрудникам, расходы на содержание животных, налоги. Данные вносятся по мере их поступления в таблицу (и потом всё сохраняется в файл). Необходимо строить ежемесячные суммарные отчёты по каждой категории расходов, вычисление операционной прибыли, рисовать график в виде круговой диаграммы.

3. Терешко

Сделать клон игры "сапёр". Игровое поле должно состоять из кнопок, размер поля должен  настраиваться.

4. Мартончик

Игра "попрыгунчик". Персонаж перемещается прыжками между платформами, которые движутся вниз. Если персонаж достигает нижней границы экрана, игра заканчивается.

5. Кизино

Игра "Scorched Earth". На форме отрисовано игровое поле с двумя танками. Танки поочерёдно стреляют друг в друга, выбирая угол и начальную скорость снаряда. 

6. Альшевский.

Создать панель быстрого запуска приложений. Запускаемые программы и их значки указываются в отдельном файле в некотором формате. При наведении мышки на значок, он увеличивается в размерах.

7. Тихонович.

При запуске на экран выводится приветствие на одном из трёх языков. По умолчанию используется язык локали. Язык можно изменить с помощью пунктов меню. Названия пунктов меню при этом тоже выводятся на выбраном языке. 

8. Бондарев.

Программа ищет установленные браузеры в системе и находит папку, где хранятся cookies. Далее пользователю предоставляется возможность удалить файлы, относящиеся к определённому сайту либо к определённому периоду времени.

9. Пунтус

Будильник/напоминалка. Программа позволяет настроить будильники/напоминалки. Будильники могут звонить однократно, либо повторно. Повторы могут быть ежедневными, ежечасными, еженедельными (в определённые дни недели), ежемесячными (в определённые дни месяца), ежегодные (в определённые даты). Для каждого будильника может быть задана мелодия (или её отсутствие) и тект мессаджбокса.

10. Самсонов

Органайзер. Пользователь ставит себе задачи на каждый день, возможно, указывая их время выполнения. При выполнении задачи, пользователь отмечает её как выполненую, может просмотреить список невыполненных и просроченных невыполненных задач.

11. Кудряшева

Диаграммы Ганта. Сотрудники компании выполняют поставленные задачи. Каждая задача загружает работой одного или нескольких сотрудников на натуральное количество дней. Вывести эти данные в виде диаграммы Ганта (не учитвать связи, оговаривающие последовательность выполнения задач). MS Project. GanttProject.

12. Хвостюк

Программа служит для проверки знаний учащихся по математике. Каждый учащийся проходит тест. Для запуска теста преподаватель вводит пароль. В тесте случайным образом генерируются задания (например на арифметические действия), выводимые с помощью label и поля ввода для ответов. Когда учащийся завершает тест,  ответы проверяются и записываются в протокол. Протокол можно просматривать, но изменять нельзя.

13. Глазунова

Задан некоторый абзац текста на произвольном европейском языке. Программа случайным образом удаляет из каждого предложения одно-два слова и выводит полученный текст на экран, вставляя на месте выброшенных слов поля для ввода. Испытуемый должен ввести в поля для ввода пропущенные слова. Пропущенные слова могут быть выведены перечнем рядом с текстом в качестве подсказки.

14. Демидович

Игра "диктатор". Вы управляете небольшой банановой республикой. В стране есть три класса граждан - рабочие, аристократы и военные. Каждый месяц вы можете удовлетворить потребности одного из классов. Удовлетворяя потребности рабочих, вы повышаете производительность труда и общий доход. Удовлетворяя потребности аристократов, вы выводите деньги на вывод денег за границу. Удовлетворяя потребности 
военных, вы снижаете риск восстания. При этом, классы, интересы которых вы не удовлетворили, становятся более недовольными. Рабочие хуже работают и поднимают восстания, Аристократы меньше платят налогов, военные могут отказать в подавлении бунтов. 

15. Филимонцев

Продажа билетов в театр. Программа позволяет выбрать сеанс и места, и заказать билеты. Каждый сеанс характеризуется названием фильма, временем начала и окончания. Данные о сеансах где-нибудь заранее хранятся. Места изображаются в виде рядов кнопок. Пользователь обозначает места, которые он хочет купить и нажимает кнопку купить. Покупка фиксируется записью в протокол (например, listbox рядом с формой). Купленные места нельзя купить повторно, обменять, сдать билеты.

16. Невейко

Фирма-продюсер организует концерты разных исполнителей и оценивает общественное мнение после проведения каждого концерта в разных возрастных группах. Программа должна позволять вносить эти данные и строить следующие отчёты: 1) для каждой группы определить возрастную категорию, где она наиболее популярна; 2) перечень самых популярных исполнителей для каждой возрастной категории; 3) перечень исполнителей, популярность которых растёт.

17. Русак

Гонки прогрессбаров. На форме распололжены несколько (можно заранее выбирать количество) прогрессбаров (индикаторов прогресса). при запуске игры, каждые 50 миллисекунд к значению Value случайно выбранномого прогрессбара добавляется случайное число от 1 до 3. При щелчке по прогрессбару к нему также добавляется единица. Цель игры - щёлкая по прогрессбарам, минимизировать разницу в их показателях. Когда один из прогрессбаров доходит до конца, игра заканчивается и игрок получает очки в сумме, равной минимальному Value среди прогрессбаров.

18. Силицкий

В книге контактов записаны сведения о людях. Для каждого человека можно добавить произвольное количество сведений - номеров телефонов, адресов, мест работы, наименований должностей. При добавлении таких сведений есть возможность выбрать тип данных из стандартного перечня выпадающим списком. Программа обладает функцией поиска, которая ищет все контакты по ключевому слову среди всех сведений о людях.

19. Скороход

Книга рецептов и меню. Программа позволяет составлять рецепты блюд, включающие перечень продуктов, и меню из этих блюд, с указанием количества порций. Для составленного меню программа вычисляет перечень необходимых продуктов с указанием их количества.
